# Real Estate ROI Predictor

A data-driven real estate analysis project designed to identify areas with strong future growth potential and return on investment.

## Idea

This project aims to analyze real estate markets using factors such as:

- Home price trends
- Rental income potential
- Nearby job growth
- Company expansion
- School quality and college readiness
- Population growth
- Crime/safety indicators
- Transportation and local development
- Overall return on investment potential

## Goal

The goal is to build a model that can score different neighborhoods or cities based on expected growth and investment attractiveness.

## Current Version

The first version of this project is a transparent scoring engine. It reads ZIP code data from a CSV file and ranks each area using a 0-100 Neighborhood Opportunity Score.

The current score combines:

- Housing momentum
- Rental ROI potential
- Job and income growth
- Population growth
- School improvement
- Vacancy, crime, and climate risk

This version does not use machine learning yet. It uses a clear weighted formula first, which makes the scoring logic easier to explain and test before adding predictive models later.

## How To Run

Install the required Python package:

```bash
pip install -r requirements.txt
```

Run the scoring engine:

```bash
python src/scoring.py
```

Run it with the Bay Area starter dataset:

```bash
python src/scoring.py data/bay_area_starter_metrics.csv
```

Run the automated tests:

```bash
python -m pytest -p no:debugging
```

Example output:

## Scoring Method

Each raw metric is normalized into a 0-100 score so different types of data can be compared fairly. Positive metrics, such as home price growth or job growth, score higher when the value is higher. Risk metrics, such as crime risk or climate risk, score higher when the risk is lower.

The current weighted formula is:

```text
Opportunity Score =
25% housing momentum
20% rental ROI
20% economic growth
10% demographic momentum
10% school improvement
15% risk score
```

Later versions can replace or support parts of this formula with machine learning predictions, such as predicted next-year home price growth.

### Census Data

The first real data collection script is `scripts/fetch_census_bay_area.py`. It pulls U.S. Census ACS 5-year population and median household income data for selected Bay Area ZIP Code Tabulation Areas, then calculates 2018-2023 growth rates.

Run it with:

```bash
CENSUS_API_KEY=your_key_here python scripts/fetch_census_bay_area.py
```

Or export the key once:

```bash
export CENSUS_API_KEY=your_key_here
python scripts/fetch_census_bay_area.py
```

Output:

```text
data/processed/bay_area_census_metrics.csv
```

Note: Census ZIP Code Tabulation Areas are ZIP-like statistical areas, not exact USPS ZIP codes.
